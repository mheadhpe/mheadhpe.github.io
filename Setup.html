The installation process can be performed without disrupting your existing infrastructure and is can be accomplished without the assistance of the Easy Connect Support team

Simplified Deployment

Mobirise

Connect power

Connect the power adapter into the power socket on your server, and connect the power adapter into your power supply.

WAN port connection

Connect the port labelled WAN into your Internet connection/router/ISP modem using an Ethernet cable. You will specify the IP address for this when you use the Installation Console.

iLO port connection

Connect the port labelled WAN into your Internet connection/router/ISP modem using an Ethernet cable. You will specify the IP address for this when you use the Installation Console.

Access The Installation Console

Access the Installation Console of your server by connecting a computer to the LAN port and by using a web browser to access the Installation Console.
